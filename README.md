# Microservices Project

## Overview
This project implements a Spring Boot microservices architecture consisting of:

- Employee Service
- Department Service

The Employee service communicates with the Department service via REST API and returns a combined response containing both Employee and Department details.

## Technologies Used

- Java
- Spring Boot
- REST APIs
- Maven
- MySQL
- Postman (API testing)
- Project Lombok (Boilerplate code reduction)

## How to Run
1. Start Department Service
2. Start Employee Service
3. Call Employee API endpoint
